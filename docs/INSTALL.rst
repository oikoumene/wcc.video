wcc.video Installation
----------------------

To install wcc.video using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``wcc.video`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        wcc.video

* Re-run buildout, e.g. with:

    $ ./bin/buildout
